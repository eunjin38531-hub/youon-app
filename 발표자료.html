진짜 사람과,

여운은 사주(四柱) 기반 데이팅 앱입니다. 생년월일·태어난 시간으로 산출한 음양오행 궁합을 신뢰의 축으로 삼아,
본인인증·AI 사진 심사로 "진짜 사람"을 검증하고, 사주 궁합과 대화 주제 추천으로 "이유 있는 만남"을 잇습니다.

음양오행 궁합
생년월일·태어난 시간으로 산출한 사주로 진짜 궁합을 계산해요. 도파민 알고리즘이 아닌 이유 있는 연결.
본인인증 검증
실명인증으로 가짜 계정 차단. AI 사진 심사로 실제 본인 사진만 등록. 진짜 사람만 남는 공간.
대화 주제 추천
사주 궁합을 기반으로 대화 주제를 제안해요. "두 분 모두 물이 많아요. 여행 얘기 어때요?"

Session Updates
이번 세션 주요 업데이트
01
"찜" → "마음에 담기"
사람을 물건 취급하는 뉘앙스 개선. 버튼·토스트·탭·카피 앱 전체 일괄 교체.
02
로그인 그라데이션 정제
Accent Coral 혼용 제거 → Primary Teal 단독. 탁하던 배경이 깔끔하게 정리됨.
03
로그인 버튼 위치 상향
엄지 닿기 편한 위치로 버튼 상향 조정. paddingBottom 40 → 80.
04
이모티콘 시스템 구축
여웅 캐릭터 30종 등록. Sticker 컴포넌트 + TS 타입 + DS 카드 추가.
05
채팅 이모티콘 전송 연동
채팅방에 이모티콘 버튼 추가. 하단 스티커 피커 UI → 선택 즉시 대화에 전송. 스티커는 채팅 버블과 별도 레이아웃으로 렌더링.
